Image Classification

ImageAnalyzer.VIP's Image Classifier classifies any image/bulk of images within seconds. The classifier uses a ResNet-50 Model trained on CIFAR-100 Dataset.
Accuracy: 93%

Object Detection

ImageAnalyzer.VIP's Object Detector detects any image/bulk of images within seconds. The classifier uses a YOLO Model trained on COCOV8 Dataset.
Accuracy: 61%
