Technologies Stack

ResNet-5​​0

50-layer convolutional neural network   (48 convolutional layers, one MaxPool layer, and one average pool layer).

SGD Optimizer

An optimization algorithm that updates model parameters iteratively using a subset of training data (mini-batch) to minimize the loss function.

Dropout & ReLu

A technique to prevent overfitting in neural networks by randomly setting a subset of activations to zero during training.

YOLOV8

An iteration of the "You Only Look Once" object detection model series, known for real-time speed and efficiency, combining advancements in architecture and training techniques to improve detection accuracy.

COCOV8

As of my last update, there isn't a widely recognized model or dataset referred to as "COCOv8"; the COCO dataset typically relates to "Common Objects in Context" used for object detection, segmentation, and captioning.

Adam Optimizer

An optimization algorithm that computes adaptive learning rates for each parameter by estimating first and second moments of the gradients.

Transfer-Learning

Transfer learning is a machine learning technique where a model developed for a specific task is reused as the starting point for a model on a second task, being enhanced for learning efficiency and reduced need for a large dataset.

Data Processing

The manipulation of data to make it suitable for analysis is referred to as data processing, involving steps such as cleaning, transforming, and aggregating data, crucial for preparing the data for effective use in business intelligence and predictive analytics.

Flask Environment

The setup in which a Flask application is run is referred to as a Flask environment, typically distinguished between development and production environments, each being configured to optimize ease of use, debugging, and performance, respectively.

Render Deployment

Cloud service where hosting is provided for websites, databases, private services, and cron jobs, facilitating app deployment with features like automatic HTTPS, pull request previews, and zero-downtime deployments.
